# вљ™ Installation

## Clone Repository

```bash
git clone https://github.com/MadasiSathwik/Task-Tracker.git
```

```
cd task-tracker
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create `.env`

```env
PORT=5000
MONGO_URI= **************************************************
NODE_ENV=production
CORS_ORIGIN=http://localhost:5173
```

Run Backend

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd frontend
npm install
```

Create `.env`

```env
VITE_API_URL=http://localhost:5000
```

Run Frontend

```bash
npm run dev
```
